# The Last Skywell Notes

## Vertical slice target

A 20–30 minute playable JRPG slice:

- Title screen
- Vael field map
- Rowan, Lyra, and Bronn as party members
- NPC dialogue
- Chest / item pickup
- Cave shrine battle
- Save/load
- Menu / inventory

## Style target

- 1993-inspired JRPG feel
- Science-fantasy, ruined sky-tech, warm villages, dangerous wilds
- Top-down movement, tile-friendly but not grid-locked
- ATB-lite combat with wait mode

## Data-first plan

Keep story, items, enemies, and skills in JSON so the game can grow without burying everything in scene code.
